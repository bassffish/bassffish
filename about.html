About This Project

This website was created as part of an IT3203 Web Development project. The purpose of this project is to demonstrate the ability to design and develop a multi-page website using HTML, CSS, and JavaScript.

The topic selected for this project is HTTP because it is one of the most important technologies behind the modern internet. Without HTTP, browsers and servers would not have a standard method of communicating.

Project Goals

  • Explain how HTTP allows communication between browsers and servers.
  • Describe the evolution of HTTP from early versions to HTTP/3.
  • Introduce important web development concepts.
  • Create an interactive quiz using JavaScript.
  • Practice designing a responsive and user-friendly website.

What I Learned

While developing this website, I learned how to organize a multi-page website, create consistent navigation, style pages with CSS, and add interactivity with JavaScript.

I also improved my understanding of how web technologies communicate and how HTTP supports everyday internet activities.
