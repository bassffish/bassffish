Key Concepts

Important terms that help explain how HTTP and web communication work.

HTTP

HTTP (Hypertext Transfer Protocol) is the system used to transfer data between web browsers and web servers on the internet.

HTTPS

HTTPS is the secure version of HTTP. It encrypts data using SSL/TLS to protect information like passwords and personal data.

Web Browser

A web browser is a program like Chrome, Safari, or Firefox that allows users to access and view websites.

Web Server

A web server is a computer system that stores websites and sends them to users when requested.

Request

A request is when a browser asks a server for a webpage or resource such as an image or video.

Response

A response is the data sent back from the server to the browser after it receives a request.

Cookies & Sessions

Cookies and sessions are used to remember user activity because HTTP itself does not store information between requests.
